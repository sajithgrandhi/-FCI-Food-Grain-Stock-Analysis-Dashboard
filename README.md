
FCI-Food-Grain-Stock-Analysis-Dashboard
Across India. Built using official FCI data from the Open Government Data platform, the project analyzes stock availability, regional contribution, and time-based trends to support data-driven understanding of national food grain management.

Overview

This project focuses on analyzing Food Corporation of India (FCI) food grain stock data for the year 2024 using Power BI. The objective of this project is to convert raw government stock data into a structured, interactive dashboard that helps users understand stock distribution, commodity trends, regional contribution, and district-level performance across India.

The dashboard enables data-driven exploration through interactive visuals, KPIs, and filters, making complex stock data easy to interpret and analyze.

Objectives
Monitor overall food grain stock levels in India
Analyze region-wise and district-wise stock distribution
Understand commodity-wise contribution and trends
Track monthly and time-based stock movement
Provide an easy-to-use dashboard for exploration and analysis
Tool & Data Source
Tool: Power BI
Dataset: Open Government Data (OGD) Platform India, Daily FCI Stocks 2024
Exploratory Data Analysis

Data cleaning and preprocessing were performed using Power BI Power Query Editor.

Cleaning and transforming the Code column to extract meaningful Region Names
Creating a custom Month Name column from date fields
Correcting spelling inconsistencies (e.g., Maharashtra)
Standardizing text case (e.g., Nagaland)
Removing unnecessary or redundant columns

These steps ensured data consistency and accuracy for reliable analysis.

Key Insights
Identification of high-stock regions and districts
Seasonal and monthly stock movement patterns
Commodity dominance across regions
District-level contribution and trends
Short-term stock projections for planning
